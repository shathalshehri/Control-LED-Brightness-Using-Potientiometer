# Arduino LED Brightness Control with Potentiometer

This project demonstrates how to control the brightness of an LED using a potentiometer and an Arduino board.

## Hardware Components

- Arduino board
- Potentiometer
- Breadboard
- LED
- 1K Ohm resistor
- Jumper wires
